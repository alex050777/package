# Api-particulier

## Prerequisites

* [node.js](http://nodejs.org)
  * [pm2](https://github.com/Unitech/pm2)
* [redis](http://redis.io/)
* [Docker](https://www.docker.com/)
  * [Docker-compose](https://docs.docker.com/compose/)

`pm2` can be installed with `npm install -g pm2` (you might need to be root)

## Install

    npm install


## Test

    npm test

## Run

    bin/www
